## Tech Stack (Backend)

- Javascript
- Express.js
- Prisma
- PostgreSQL
- bcryptjs (JWT)
- morgan

## Tech Stack (Frontend)

- Typescript
- Next.js
- Next Auth
- Redux
- Tailwind CSS
- zod


## How To Clone

### Step 1
```
git clone https://github.com/Akbaroke/simple-ecommerce-byakbaroke
```

### Step 2
```
cd ./simple-ecommerce-byakbaroke
```

### Step 3
```
cd ./backend
```

### Step 4
```
npm install
```

### Step 5
- create an `.env` file by copying from the `.env.example` file
- edit the database connection in the `DATABASE_URL` variable

### Step 6
```
npm run start
```

### Step 7
Open New Terminal
```
cd ./frontend
```

### Step 8
```
npm install
```

### Step 9
- create an `.env` file by copying from the `.env.example` file
- copy the `TOKEN_SECRET` variable from the backend `.env` file

### Step 10
```
npm run dev
```

### Last Step
- Open in browser http://localhost:3000
